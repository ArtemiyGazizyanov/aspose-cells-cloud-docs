## **Introduction**
This example shows how to remove all images from a Excel Worksheet using Aspose.Cells Cloud API.
## **API Information**

|**API**|**Type**|**Description**|**Resource Link**|
| :- | :- | :- | :- |
|/cells/{name}/worksheets/{sheetName}/pictures|DELETE|Delete all images from a Excel Worksheet|[DeleteWorkSheetPictures](https://apireference.aspose.cloud/cells/#/Pictures/DeleteWorkSheetPictures)|
### **cURL Example**
{{< tabs tabTotal="2" tabID="1" tabName1="Request" tabName2="Response" >}}

{{< tab tabNum="1" >}}

```java

curl -X DELETE "http://api.aspose.com/v1.1/cells/Sample_Test_Book.xls/worksheets/Sheet6/pictures" -H "Content-Type: application/json" -H "Accept: application/json"

```
