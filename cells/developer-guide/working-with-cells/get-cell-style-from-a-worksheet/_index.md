## **Introduction**
This example shows how to read Cell Style information from a Excel Worksheet using Aspose.Cells Cloud API.
## **API Information**

|**API**|**Type**|**Description**|**Resource Link**|
| :- | :- | :- | :- |
|` `/cells/{name}/worksheets/{sheetName}/cells/{cellName}/style|` `GET|Read Cell Style information from a WorkSheet|` `[GetWorksheetCellStyle](https://apireference.aspose.cloud/cells/#/Cells/GetWorksheetCellStyle)|
### **cURL Example**
{{< tabs tabTotal="2" tabID="1" tabName1="Request" tabName2="Response" >}}

{{< tab tabNum="1" >}}

```java

curl -X GET "https://api.aspose.cloud/v3.0/cells/test.xlsx/worksheets/Sheet1/cells/a1/style" -H "accept: application/json" 

```
